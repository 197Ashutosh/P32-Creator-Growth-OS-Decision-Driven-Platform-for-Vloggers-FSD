# 🚀 Creator Growth OS — Decision-Driven Platform for Vloggers

## 📌 Overview

Creator Growth OS is a full-stack decision-support system designed to help vloggers make data-driven decisions. Unlike traditional analytics tools, this platform transforms raw performance data into clear, actionable recommendations.

---

## 🎯 Problem Statement

Content creators often struggle to interpret analytics from platforms like YouTube Studio. While data is available, it does not provide clear guidance on what actions to take next, leading to inconsistent growth and decision-making.

---

## 💡 Solution

This system bridges the gap between data and decision-making by analyzing creator performance and providing:

* Actionable next-step recommendations
* Content strategy insights
* Consistency tracking
* Performance-based suggestions

---

## ⚙️ Features

* 📊 Smart performance tracking
* 🧠 Rule-based decision engine
* 🎯 Next-action recommendation system
* 🎮 Gamification (streaks, consistency score)
* 🧪 Experimentation module

---

## 🏗️ System Architecture

* **Frontend:** React
* **Backend:** Node.js + Express.js
* **Database:** MongoDB Atlas
* **Deployment:** AWS (EC2, S3, CloudFront)

---

## 🔁 Workflow

1. User inputs content data
2. System stores and processes data
3. Pattern analysis is performed
4. Decision engine generates insights
5. User receives actionable recommendations

---

## 🧠 Key Concept

This project focuses on **decision-making**, not just analytics.
It answers the critical question: **“What should the creator do next?”**

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/creator-growth-os.git
cd creator-growth-os
```

### 2. Setup Backend

```bash
cd server
npm install
npm start
```

### 3. Setup Frontend

```bash
cd client
npm install
npm start
```

---

## 🔐 Environment Variables

Create a `.env` file in the server directory:

```env
MONGO_URI=your_mongodb_connection_string
PORT=5000
```
